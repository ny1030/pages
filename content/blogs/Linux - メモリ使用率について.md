# Linux - メモリ使用率について
## freeコマンド
- total : 合計
- used : カーネルとプロセスが使用している
- shared : tmpfsに使われている
- free : 余っている
- buffers : バッファキャッシュのメモリサイズ
- cache : ページキャッシュのメモリサイズ
- available : 実質的な空きメモリ
	- free + buff/cache (解放可能な部分)
